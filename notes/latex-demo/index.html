1. Inline and Display Math

Inline: the Gaussian distribution is $p(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$. You can also use custom macros: the norm $\norm{v}$ and the inner product $\inner{u}{v}$.

Display mode centers the equation and gives it more breathing room:

$$ \mathcal{L}(\theta) = -\sum_{i=1}^{N} \log p_\theta(x_i) $$

2. Numbered & Labeled Equations

Equations are auto-numbered when you use \begin{equation}. Reference them with anywhere in the note.

\begin{equation} \label{eq:bayes} p(\theta \mid x) = \frac{p(x \mid \theta)\, p(\theta)}{p(x)} \end{equation}

Bayes' theorem \eqref{eq:bayes} is the foundation of probabilistic inference.

3. Aligned Systems

Use align to line up multi-line derivations on a symbol (typically =):

\begin{align} \nabla_\theta \mathcal{L} &= -\sum_{i=1}^{N} \nabla_\theta \log p_\theta(x_i) \\ &= -\sum_{i=1}^{N} \frac{\nabla_\theta p_\theta(x_i)}{p_\theta(x_i)} \\ &= -N\, \E_{x \sim p_\theta}\!\left[\nabla_\theta \log p_\theta(x)\right]. \end{align}

5. Callout Boxes

Definition
A function $f: \R^n \to \R$ is convex if for all $x, y \in \R^n$ and $\lambda \in [0,1]$, $$ f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y). $$
Theorem (Jensen's Inequality)
If $f$ is convex and $X$ is a random variable, then $f(\E[X]) \leq \E[f(X)].$
Remark
Equality holds iff $X$ is constant a.s. or $f$ is affine.
Warning
Do not confuse the empirical risk $\hat{\mathcal{L}}_N$ with the population risk $\mathcal{L}$; the gap matters when $N$ is small.

6. Code Blocks

Use fenced code blocks for algorithms and snippets (highlighted via highlight.js):

import numpy as np

def gradient_descent(f_grad, x0, lr=1e-3, steps=1000):
    """Vanilla gradient descent."""
    x = x0.copy()
    for _ in range(steps):
        x -= lr * f_grad(x)
    return x

# Sampling from a multivariate Gaussian
mu = np.zeros(2)
Sigma = np.array([[1.0, 0.8],
                  [0.8, 1.0]])
samples = np.random.multivariate_normal(mu, Sigma, size=1000)

7. Custom Macros

The MathJax config at the top of this file defines several handy macros. Add your own in the macros block of the MathJax config:

  • \R → $\R$  (reals)
  • \E → $\E$  (expectation)
  • \norm{v} → $\norm{v}$
  • \abs{x} → $\abs{x}$
  • \inner{u}{v} → $\inner{u}{v}$
